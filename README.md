jquery-hemi-custom-select
=========================

jQuery custom select plugin. Custom responsive design.

How to use:
-----------

```html
<select class="form-control js-hemi-custom-select2">
	<option>1</option>
	<option>2</option>
	<option disabled="">3</option>
	<option>4</option>
	<option>5</option>
</select>
```

```script
$(function () {
	$(".js-hemi-custom-select1").hemiCustomSelect({
	});
});
```
